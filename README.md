# Ranking F1 do Servidor

Site estatico para acompanhar o campeonato de F1 do servidor: rankings da Serie A ate a Serie G, proximas corridas, noticias, dados das lives e regras.

## Como editar

- Altere pilotos, pontos, corridas, noticias, lives e regras em `data.js`.
- Ajustes visuais ficam em `styles.css`.
- A pagina principal fica em `index.html`.
- Cada secao do menu tem sua propria pagina: `rankings.html`, `corridas.html`, `noticias.html`, `lives.html` e `regras.html`.

## Publicacao

Este projeto foi preparado para GitHub Pages usando arquivos estaticos na branch `main`.
